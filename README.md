## Instalacion

### 1. Clonar repositorio

```shell
git clone https://github.com/sebabrennan/romapastasReact-Brennan.git
```
### 2. Ir al proyecto

```shell
cd roma-pastas
```

### 3. Instalar dependencias

```shell
$ npm install
```
### 4. Correr el proyecto

```shell
$ npm run start
```
